hardware

design =>it will be modular  ip68-69   defined size intervals   it will be flexible like  if module is 60 device space is currently 50  user pushes all sides and out module there so it's locked in, communication can be via gold pins or other similar stuff with option to company specific communication too like if part a and b same company no need universal language or pins for that scenario so it can be global+custom modules,    phone itself can expand too, flexible+rigid,  like if battery planned is big expand frame put battery, push till hearing auto click or manual specific position lock,  flexing can be via complex stuff or just spring like things(making sure heat dissipation is working nice like being careful about air which is heat insulator, flex pressure fix edges but inside needs proper design),  also some expander modules possible too,  letting user do whatever weird thing they desire( like adding 10 different camera modules or audio etc each with different spectrum, frequency etc) while also making future proof like if some reason something isn't supported you can put adapter like stuff etc

production=> since basics are defined, universal and public everyone can produce each piece and use each piece, so if you're a bad camera producer just ship with known nice module put there or add option without it so users can buy without it and add themselves or using from their older device,    since you can focus more narrow and since others also attempt similar thing and some stuff are open you can use that progress to fasten your narrow tech and ship it,  like similar to no need Intel just to add monitor logic ( you still need to make compatible with universal language with option to add custom stuff if you make more than one module yourself or some group collective desicion)   similar to type c producers, for almost all devices, so if you make overall bad thing buy your camera is awesome you can mass sell that module too

 also micro/extra payment is allowed too like if agreement is 100w and if producer crack some stuff and make it 120 by software there can be option like pay this for extra 20,   for example it can be something like going to a vacation etc where it's needed temporarily

security=> factory is first user with ownership, if owner keeps logging on  modules records each frame they are connected to as read only with only delete option or corresponding line if its logs of current owner,  so no steal and erase thing,

      it comes with minimum 8 values     hardware id owner id,  transfer on off, selling on off, transportation on off, stolen on off, antitheft on off, logging on off,      
  user can buy from factory with their id already written or from local store  with transportation turned off  while selling and transfer on,  
  selling requires handshake if both sides agree(transportation flag can be also used for that), it can be over a trusted server so modules can work at other device that is not owner,  
  after selling owner id changes while keeping history if logging on so easier to solve if there is malicious actions(logging is read only, it can only expand(pointers can be used to follow stack)),   
  if bought from second hand store  and if  transfer off it prompts requirement of putting in original device else connect to server for verification where it check reports of stolen devices and if owner declares transfer on online etc ( like when if original device is not working properly but owner have consent) ( no offline solution if anti theft on original device broken transfer off etc at once)   
  if device stolen user can simply declare it's stolen so itself and pieces are useless, thus no incentive to steal it( unless anti theft off transfer on etc at the same time)(it also alerts police etc if declared stolen so it's even risky to check it unless owner itself declared it's someone known who found it etc)(if it's just lost a reward can be added too like if you find my camera you get 50 kinda stuff)
  users can freely send a module to a friend like for example giving better battery and camera to a friend going concert while taking their hi power sound module for work presentation or projector module etc,  just turn transfer on  selling off transportation off  and give the piece, if piece got lost simply declare stolen
